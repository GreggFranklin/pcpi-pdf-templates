# PCPI PDF Templates

Custom Gravity PDF templates for PCPI workflows.

## Installation

1. Upload plugin to `/wp-content/plugins/`
2. Activate plugin
3. Templates will automatically register

## Structure

Each template lives in its own folder:

templates/{template-name}/

## Notes

- Template filename must match preview image
- Designed for integration with PCPI Workflow Engine
